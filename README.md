# era-2.0-webui
ERA 2.0 Web User Interface Prototype
+ JSP 2.0
+ JSTL 1.2

## Building & running era-2.0-webui from source
System Requirements
+ Java 6
+ Maven 3
```shell
git clone https://github.com/ualbertalib/era-2.0-webui.git
cd era-2.0-webui
mvn jetty:run
```
Point your browser to http://localhost:8080
